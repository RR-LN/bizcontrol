# BizControl 360 - Projeto

## O QUE ESTOU CONSTRUINDO?
Sistema para lojas controlarem estoque, vendas e lucro sem ter que usar papel.

## QUAIS PARTES TEM?
1. **Painel do Dono** (vê tudo)
2. **Caixa** (faz vendas)
3. **Estoque** (cadastra produtos)

## REGRAS DE OURO:
- NUNCA apague a pasta `node_modules`
- SEMPRE que funcionar, faço "commit" (salva versão)
- Se der erro vermelho, copio e colo no ChatGPT
